# Brisk Parser

Drop-in, multi-purpose parser


## Dependencies

* [Body parser](https://github.com/expressjs/body-parser)
* [Busboy](https://github.com/mscdex/busboy)


## Install

Using npm
```
npm install brisk-parser
```
